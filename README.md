# Customer Intelligence & Churn Prediction Platform

## Project Overview

This project is an AI-powered Customer Churn Prediction Platform developed using Google Cloud technologies. The system analyzes customer behavior patterns, predicts customer churn using machine learning, visualizes business insights through dashboards, and generates AI-powered recommendations.

The objective of this project is to identify customers who are likely to leave and help businesses implement retention strategies.

---

## Problem Statement

Customer churn directly affects business growth and revenue. Businesses often struggle to identify customers at risk before they leave. This project helps predict churn probability using customer behavior data and provides actionable business recommendations.

---

## Technologies Used

- Google BigQuery
- BigQuery ML
- SQL
- Looker Studio
- Vertex AI
- Google Cloud Platform (GCP)

---

## Dataset Features

The dataset contains the following customer attributes:

- CustomerID
- Age
- Gender
- Tenure
- Usage Frequency
- Support Calls
- Payment Delay
- Subscription
